Mis Proyectos de Python - CFT Estatal
Este repositorio contiene herramientas de automatización desarrolladas en Python:

Limpiador.py: Script para eliminar archivos temporales y optimizar espacio.

Respaldo.py: Automatización de copias de seguridad de carpetas importantes.

Soporte.py: Herramienta de asistencia técnica para hardware y software.
